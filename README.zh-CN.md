# TimePeek

TimePeek 是一款 macOS 菜单栏小工具，会在你复制时间戳或日期字符串时自动识别，并以浮窗形式即时显示可读的时间转换结果。

[App Store](https://apps.apple.com/cn/app/timepeek/id6757980339?mt=12)

## 功能特性
- 菜单栏常驻，轻量可随时使用
- 自动识别秒/毫秒时间戳与常见日期格式
- 即时展示 UTC、本地时区与常用时区的转换结果
- 一键复制转换值
- 可自定义日期格式、自动隐藏时间与语言

## 使用方式
1. 从 Xcode 启动应用。
2. 复制一个时间戳或日期字符串。
3. 浮窗会自动出现并显示转换结果。
4. 点击菜单栏图标打开设置。

## 构建
1. 用 Xcode 打开 timepeek.xcodeproj。
2. 选择 timepeek scheme。
3. 运行。

## 本地化
TimePeek 提供英语、简体中文、日语与葡萄牙语（巴西）的界面语言。
