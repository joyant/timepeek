# TimePeek

[App Store](https://apps.apple.com/cn/app/timepeek/id6757980339?mt=12) · [中文说明](README.zh-CN.md)

TimePeek is a macOS menu bar utility that detects timestamps or date strings on your clipboard and instantly shows readable conversions in a floating bubble.

## Features
- Menu bar app with a lightweight, always-available workflow
- Clipboard monitoring for Unix timestamps (seconds or milliseconds) and common date formats
- Instant conversions with UTC, local time, and a favorite timezone
- One-click copy for any converted value
- Customizable date format, auto-hide timing, and language

## Usage
1. Launch the app from Xcode.
2. Copy a timestamp or date string.
3. A bubble appears with conversions you can copy.
4. Use the menu bar icon to open Settings.

## Build
1. Open timepeek.xcodeproj in Xcode.
2. Select the timepeek scheme.
3. Run.

## Localization
TimePeek includes English, Simplified Chinese, Japanese, and Portuguese (Brazil) UI strings.
